## 오늘의 메모

> 간단한 메모나 ToDo List를 작성하고 동영상이나 이미지를 기록할 수있는 Web
> Application 입니다.

<h3> ☝🏼 순수 HTML과 CSS를 이용하여 UI를 구현 하였습니다. </h3>

<h3> ☝🏼 TypeScript를 이용하여 객체지향적으로 구현하였습니다. </h3>

<h3> ☝🏼 Web API를 이용하여 메모를 드래그 앤 드롭하는 기능을 구현 하였습니다. </h3>

<br>
