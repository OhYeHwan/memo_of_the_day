## 프로젝트 소개 🎉

<br>

**"오늘의 메모"** 는 간단한 메모나 ToDo List를 작성하고 동영상이나 이미지를 기록
할 수있는 웹 애플리케이션 입니다.

<br>

<h4> ☝🏼 순수 HTML과 CSS를 이용하여 UI를 구현 하였습니다. </h4>

<h4> ☝🏼 TypeScript를 이용하여 객체지향적으로 구현하였습니다. </h4>

<h4> ☝🏼 Web API 중 드래그 앤 드롭 이벤트들을 이용하여 메모를 드래그 앤 드롭하여 위치
를 변경하는 기능을 구현 하였습니다. </h4>
